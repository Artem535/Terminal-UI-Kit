=== Added

- LogView component with follow/pause and severity badges (#PR7B)
- streaming_log_viewer interactive example application
- render_styled_text utility for StyledText-to-FTXUI Element conversion

* Repository foundation: CMake + Xmake build systems, vcpkg manifest,
  directory skeleton for `include/`, `src/`, `tests/`, `examples/`,
  `benchmarks/`, and `docs/`, code style configuration (clang-format,
  clang-tidy), CI workflow skeleton, and governance documents.
* Testing foundation: GoogleTest/CTest wiring and a virtual-screen
  rendering-test helper (`terminal_ui_kit::test_support::render_to_text`).
* Core Text Model: `TextPosition`, `TextRange`, `TextStyle`/`Color`,
  `StyledText`/`TextSpan`, and a UTF-8 codepoint-safe `wrap_plain_text()`
  word-wrap utility.
* Documentation site: AsciiDoc content published via Antora, replacing the
  Markdown docs that previously lived only in the repository root.
* Variable-height `VirtualList` support with deterministic height estimates,
  measured layout caching, prefix-sum range lookup, and scroll anchoring when
  visible rows change height.
* Streaming document and log model support: chunk-safe UTF-8 ingestion,
  ANSI SGR parsing into `StyledText`, retained log entries with severity and
  substring filtering, and revision tracking for incremental consumers.
  The interactive `LogView` and streaming log viewer example are planned for
  PR7B.
